Luminous

by Rebecca Sweigart


This project was created during the NYU IMA Low Res Summer intensive in New York City.

What:
A mirrored acrylic container with three buttons that control the output on p5.js.

Why:
I wanted to make an interactive 3D container that creates 2D art.

How:
1.) Laser cut mirrored acrylic for the container
2.) Arduino connects the button input to the p5 canvas
3.) P5 Reads the inputs - the yellow button outputs a different color, the blue button outputs larger stars, and the red starts the stars again




Note: the above p5.js sketch will only work when the arduino is plugged into the user's computer.
